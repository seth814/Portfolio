TensorFlow Object Detection API

Description:

I wanted to showcase how the TensorFlow Object Detection API can be used to create a custom localization model. I used the ssd_mobilenet_v1_fpn model and re-trained it to detect four classes (eye, nose, mouth, ear). The model was trained for 1 hour using 300 images of my face. Results were quite impressive, and could get even better with more training images.
